### Symbols and units
| Symbol      | Description | Units       |
| ----------- | ----------- | ----------- |
| $\lambda$   | geocentric longitude of the Moon's centre| |
| $\beta$     | geocentric latitude of the Moon's centre | |
| $\Delta$    | distance between the center of the Earth and Moon | km |
| $\pi$       | equatorial horizontal parallax of the Moon | |
| T           | dynamical time | centuries|
| L'          | Moon's mean longitude | degrees |
| D           | Mean elongation of the Moon | degrees |
| M           | Sun's Mean anomaly | degrees |
| M'          | Moon's mean anomaly | degrees |
| F           | Moon's argument of latitude| degrees |
